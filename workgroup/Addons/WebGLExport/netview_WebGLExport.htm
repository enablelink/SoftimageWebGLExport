WebGL Export

The WebGL Export allows you to export a full scene to a WebGL webpage.

Supported features

  • Polygon mesh (max 64.000 vertex indices)
  • Directional light (one)
  • Phong materials (diffuse, specular and ambient)
  • Diffuse textures (png and jpeg only)
  • Perspective camera

Workflow

Before exporting the scene make sure that all all visible objects are converted into polygon mesh (otherwise they will not be exported). Also make sure that the scene contains one directional light source.

All materials used in the scene must be Phong materials.

Only one texture is supported for each object. Make sure that each object only has a single texture projection.

When the scene is ready for export, select the WebGL export wizard from the WebGL export menu.

If needed you can change the subdivision level and construction mode from the geometry group. Most often the default values will be the best choice.

Select the export path and press Export

Running the example

To run the example a browser with WebGL support is needed. At the time writing the following browsers have WebGL support:

  • Firefox 4.0 (or newer)
  • Chrome 9.0 (or newer)

The exported project requires that it running on a web-server. But to make it easy to test, a simple web-server is bundled with the export.

To launch the local web-server, simple double-click the SimpleWebServer.jar (Java 6 needs to be installed)

To start the webgl page, go to the web page http://localhost:5000/index.html or http://127.0.0.1:5000/index.html in your browser with the SimpleWebserver running.

To run on a webserver the, simply put all the exported files on a webserver and navigate to the index.html from a browser.
